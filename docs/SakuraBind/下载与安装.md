# 下载与安装

下载插件并安装到你的服务器

## 环境依赖
 * JRE 1.8 及以上
 * Minecraft 1.8 及以上
 * 使用 sipgot-api 的服务端
 * 第一次使用插件将会**联网**下载依赖

## 安装插件
:::danger[注意]
**请不要热加载插件!!!**
:::
将插件放入 服务端 plugins 目录，启动服务端
